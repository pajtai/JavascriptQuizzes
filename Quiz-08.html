Quiz 8:

Consider the code at the bottom of the page... (uses jQuery in places not relating to the question)

The program will print two lines.
  • What three words will line one print?
  • Which branch of the if statement will line two show?
    • Explanation: This behavior is caused by type conversion. "new Boolean(false)" is an Object, but when it is used in a Boolean context (if ...) then this Object is converted to a primitive Boolean. Any Object converted to a primitive Boolean is always "true"..... even it it's internal value is "false"!
    $(function() {
    var truth = new Boolean(false);
    $("body").append(truth + " is " + (truth ? "true" : "false") + ".<br/>");
    if (truth) {
        $("body").append("Yup, it's true");    
    } else {
        $("body").append("Nope. It's false.");    
    }
});

Quiz 15:

Consider the code below
var x1 = 1;
x2 = 2;

delete x1;
delete x2;

alert ("x1: " + (x1 || "naught") + " x2: " + (x2 || "naught"));
  • What error will the code produce?
  • What is the word (which will be followed by a period in the code) that can fix this error?
  • Once this fix is made, what would the output of the program be?
  1. The error will be: "x2 is not defined"
  2. The word is "window". This is because, if "var" is ommited, x2 becomes a "global variable" by becoming a property of "window". If you use "var" in the global scope, this does not happen. So omitting "var" is not the same as using "var" in the global scope.
  3. To fix the code, you could replace 'x2 || "naught"' with 'window.x2 || "naught"'. This is the program output that is shown if you click on "Toggle Fixed Program Output" above.

Quiz 14:

Consider the code below 
(function() {
    var a = 1,
        fun = function () {
            return function() { console.log(this.a); };
        },
        b = fun();
    b.a = 5;   
    b();
}());
  • What will the program output?
  1. In Javascript functions, "this" refers to the owner of a function.
  2. Objects defined in the global scope are owned by "window".
  3. The function returned by "fun" was defined without var, so it is in the global scope. It is a function declaration. Functions are Objects.
  4. Calling b.apply(b) would output 5
  5. Setting "window.a = 7;" before "b();" would output 7
  6. Using "console.log(a)" would output 1

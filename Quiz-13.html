Quiz 13:

Consider the code below 
console.log((!NaN+[])[+!{}]);
  • What will the program output?
  1. ("true")[0] is "t"
  2. {} is truthy, since any object is.
  3. so, !{} is false
  4. +!{} transforms false to a Number by using the Unary Plus Operator - the number is 0
  5. NaN is falsey, so !NaN is true
  6. !NaN+[] concatenates "true" and "", since the String representation of an empty Array is an empty String
  7. Throw all of that together and we are accessing the letter at index 0 of the string "true"
